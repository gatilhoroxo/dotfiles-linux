# dotfiles-linux
Legendery fiesta com Ubuntu e outros.
Configuração pessoal de ambiente Ubuntu.

## O que está aqui
- **distros**: `.bashrc` e módulos `.bash_config/` (aliases, funções, variáveis) para algumas distros
- **personalization**: Scripts para baixar sites offline e configurar IAs locais

## Instalação
```bash
git clone https://github.com/gatilhoroxo/dotfiles-linux 
cd ~/dotfiles-linux && bash scripts/install.sh
```

## Uso após instalar
| Comando          | O que faz                        |
|------------------|----------------------------------|
| `beginUser`      | Configura novo desktop Ubuntu    |
| `sysupdate`      | Atualiza todo o sistema          |
| `awake <min>`    | Mantém tela acordada por X min   |
| `rustbook-start` | Sobe servidor local do Rust Book |

## Estrutura

distros - ubuntu24 e mint22
personalization - local stuff
